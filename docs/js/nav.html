lux.nav¶

Angular module for site navigation.

navbar¶

Directive which create a bootstrap navbar:

<navbar data-options='examples_navbar'></navbar>
<script>
    var examples_navbar = {
        brand: 'Navbar',
        items: [{href: '#', name: 'Link1'},
                {href: '#', name: 'Link2'}],
        itemsRight: [{href: '#', name: 'Link3'},
                     {href: '#', name: 'Link4'}],
    };
</script>
