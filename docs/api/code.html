Highlight Code¶

Highlight code snippets with highlightjs.

Required extensions: lux.extensions.ui

Usage¶

Include lux.extensions.code into the EXTENSIONS list in your config file:

EXTENSIONS = [
    ...
    'lux.extensions.ui',
    'lux.extensions.code'
    ...
    ]

check the code hightlight angular module for more information.
