Static site builder¶

Static site generator

Usage¶

Put the lux.extensions.static extension into your EXTENSIONS list and build the static web site via the build_static option in the command line:

python managet.py build_static

There are several parameters which can be modified in order to customise the site.

The first step is to create a new project via the create_project

When creating the middleware for the static site one can pass the following key-valued parameters during initialisation of a HtmlContent, a Blog and other routers:

  • dir the directory where the source files are located, if not provided, it is assumed it is the same as the route parameter
  • html_body_template to specify the location of the template which render the html body tag of the Html5 document. If not provided, the parent router value is used unless no parent is available and the STATIC_TEMPLATE is used instead.
