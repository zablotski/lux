Authentication¶

A lux extension for managing users, sessions and permissions. The extension is added by inserting lux.extensions.sessions into the list of EXTENSIONS of your application.

There are several parameters which can be used to customise authorisation.

class lux.extensions.sessions.backend.AuthBackend(app)[source]¶

Interface for authentication backends.

An authentication backend manage authentication, login, logout and several other activities which are required for managing users of a web site.

During a client request, the instance of the authentication backend can be accessed from the request cache dictionary:

backend = request.cache.auth_backend
