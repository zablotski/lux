Dialog¶

To create a dialog on the fly:

var dialog = lux.web.dialog({closable: true});

Available Options¶

dragdrop¶

An optional Drag and drop object to manage drag and drop functionalities. If this is given the dialog will be draggable and it can be dropped into other containers.
