Drag & Drop¶

HTML5 extensions for dragging and dropping elements.

Available Options¶

dropzone¶

Can be a selector or a jQuery element. If not available it is assumed the dropzone is the whole html body.

dragenter¶

dragenter(draggable, target, e)

Handle the dragenter event e when the draggable enters a target element from the dropzone selector.

drop¶

drop(draggable, target, e)

Handle the drop event e when the draggable is dropped into target.
