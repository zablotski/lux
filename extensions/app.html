The Application¶

Parameters¶

CSS¶

Default: '{}'

Dictionary of css locations.

DATE_FORMAT¶

Default: '%Y %b %d %H:%M:%S'

Default formatting for dates

DEFAULT_TEMPLATE_ENGINE¶

Default: 'python'

Default template engine

DESCRIPTION¶

Default: 'None'

A description to display before the argument help on the command line when running commands

ENCODING¶

Default: 'utf-8'

Default encoding for text.

ERROR_HANDLER¶

Default: 'None'

Handler of Http exceptions

EXTENSIONS¶

Default: '[]'

List of extension names to use in your application. The order matter since the wsgi middleware of extension is processed in the order given by this setting.

HTML_HEAD_TITLE¶

Default: 'Lux'

Default HTML Title

HTML_META¶

Default: "[{'content': 'IE=edge', 'http-equiv': 'X-UA-Compatible'}, {'content': 'width=device-width, initial-scale=1', 'name': 'viewport'}]"

List of default meta elements to add to the html headelement

JSREQUIRED¶

Default: "('jquery', 'lodash', 'json', 'lux')"

Default Required javascript. Loaded via requirejs.

JSREQUIRE_CALLBACK¶

Default: 'lux.initWeb'

Callback used by requirejs.

LINKS¶

Default: "{'python': 'https://www.python.org/', 'lux': 'https://github.com/quantmind/lux', 'pulsar': 'http://pythonhosted.org/pulsar/index.html'}"

Links used throught the web site

LOGGING_CONFIG¶

Default: 'None'

Dictionary for configuring logging

MEDIA_URL¶

Default: 'media'

the base url for static files

MINIFIED_MEDIA¶

Default: 'True'

Use minified media files. All media files will replace their extensions with .min.ext. For example, javascript links *.js become *.min.js

SITE_URL¶

Default: 'None'

Web site url
